Android OSINT-кабинет по открытым источникам.

- Пробив телефона/email: ссылки на открытый поиск ФИО и следов утечек (Google/Yandex/Truecaller web и др.)

- Instagram/X: открытие в браузере, жёстче детект; bot-protected выкл. по умолчанию
