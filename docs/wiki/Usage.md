# Usage

Тёмный кабинет: режимы сверху, поле запроса, карточка отчёта, журнал дел.

![Workbench](https://raw.githubusercontent.com/Hanter1/Sherlock/main/docs/assets/mock-workbench.png)

## Режимы

| Режим | Что делает |
|-------|------------|
| **Никнейм** | Параллельный скан по каталогу площадок |
| **Телефон** | Нормализация + оператор; WhatsApp/Telegram/Truecaller; Google/Yandex (ФИО и открытые следы утечек) |
| **Email** | MX + SPF/DMARC (DoH) + Gravatar; HIBP/поиск по открытому индексу |
| **ФИО** | Ники из ФИО (транслит) → скан как Sherlock + ссылки Google/Yandex/VK |

## Команды

| Команда | Действие |
|---------|----------|
| `/username nick` | Скан ника |
| `/username a b c` | Очередь до **5** ников подряд |
| `/compare a b` | Сравнение двух ников |
| `/clear` | Очистка журнала |
| `/about` | Версия APK и каталогов |

Также можно вводить запрос без слэша в выбранном режиме.

## Отчёт по нику

![Report](https://raw.githubusercontent.com/Hanter1/Sherlock/main/docs/assets/mock-report.png)

- Сводка: найдено / неуверенно / нет / ошибки
- Фильтры и группировка по категориям (dev, social, …)
- **Стоп** — отмена mid-scan
- **Добить ошибки** — повтор только ERROR + UNCERTAIN
- **Повторить без кэша** — принудительный рескан (+ Δ с прошлого)
- HTTP-диагностика: код, редиректы, `rate limited`, причина UNCERTAIN/ERROR
- Экспорт Markdown / JSON (`confidence`: confirmed / uncertain)
- Share / copy с предупреждением при ПДн и опцией маскирования

## Журнал

- История на устройстве (EncryptedFile); опция «не сохранять»
- Поиск по истории (лупа)
- «Дела» по дням, удаление одного дела
- Закрепление последнего отчёта

## Кэш

- Сессия + диск, TTL **24 ч**
- В отчёте и настройках — возраст / остаток TTL

## Уведомления

Если скан закончился в фоне: «скан … готов · N найдено».
