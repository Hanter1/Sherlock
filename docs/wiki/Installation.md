# Installation

## Требования

- Android **8.0+** (API 26)
- Интернет для скана ника и email (телефон / ФИО работают и офлайн на уровне локальной логики + ссылки)

## APK из GitHub Releases

1. Откройте [Releases](https://github.com/Hanter1/Sherlock/releases/latest).
2. Скачайте `Sherlock-Bot-*-debug.apk` (или актуальный артефакт из notes).
3. Разрешите установку из неизвестных источников для браузера/файлового менеджера.
4. Установите APK.

> Debug-сборка подписана debug-keystore. Для production нужен свой release keystore — см. [Building](Building).

## Разрешения

| Разрешение | Зачем |
|------------|--------|
| Интернет | HTTP-проверки площадок, MX/DoH, Gravatar, remote-каталог |
| Уведомления (`POST_NOTIFICATIONS`) | Сообщение о конце скана, если ушли из приложения |

Приложение может скрывать содержимое в «Недавних» (`FLAG_SECURE`) — опция из настроек/политики приватности UI.

## Первичная настройка

1. Примите дисклеймер при первом запуске.
2. При необходимости включите уведомления в системных настройках Android.
3. В настройках приложения: пресет площадок, параллелизм, email-согласия, remote-каталог.
