# Privacy and Ethics

## Границы продукта

Sherlock Bot — клиент OSINT по открытым HTTP/DNS-источникам и публичному поиску.

Приложение **не**:

- не подключается к закрытым базам и платным «пробив»-API
- не обходит аутентификацию чужих аккаунтов

Приложение **да**:

- ищет публичные профили по нику
- для телефона/email даёт открытые ссылки (мессенджеры, Truecaller web, Google/Yandex), по которым в индексе могут быть ФИО и уже опубликованные следы утечек

## На устройстве

- История может шифроваться на диске (`EncryptedFile`)
- Опция не сохранять историю
- `FLAG_SECURE` — опциональное скрытие превью в «Недавних»
- Опция маскировать телефон/email при share/copy
