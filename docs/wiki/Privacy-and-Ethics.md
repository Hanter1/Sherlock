# Privacy and Ethics

## Границы продукта

Sherlock Bot — клиент **легального OSINT** по открытым HTTP/DNS-источникам.

Приложение **не**:

- не подключается к закрытым базам и «сливам»
- не обходит аутентификацию чужих аккаунтов
- не предназначено для преследования, мошенничества или незаконного сбора ПДн

## На устройстве

- История может шифроваться на диске (`EncryptedFile`)
- Опция не сохранять историю
- `FLAG_SECURE` — скрытие превью в «Недавних»
- Share/copy предупреждает при персональных данных; можно маскировать телефон/email

## Ответственность

Используйте только для законных целей и с уважением к приватности. Авторы не несут ответственности за злоупотребление инструментом.

При первом запуске показывается дисклеймер — его нужно принять для работы.
