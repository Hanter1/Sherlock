# Sherlock Bot v1.15.0

Android OSINT-кабинет по открытым источникам.

## Download

- **Sherlock-Bot-1.15.0-debug.apk** — debug-сборка (API 26+).

## What's new

- Каталог **~480 площадок** из [sherlock-project/sherlock](https://github.com/sherlock-project/sherlock) (MIT) + curated
- Детект `status_code` / `message` / `response_url`
- Пресеты **Быстрый** и **Sherlock Full**; NSFW выкл. по умолчанию
- Убраны блокирующие дисклеймер / email-consent / ПДн-диалоги при шаринге
- UI: компактный отчёт, короткие ссылки, скрытие клавиатуры после ввода
